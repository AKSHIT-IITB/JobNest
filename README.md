# JobNest

JobNest is a web-based job portal built using the **MERN stack**.  
It provides a simple platform where recruiters can post job openings and job seekers can browse and apply for relevant roles.

The project focuses on clean architecture, basic authentication, and practical use of full-stack web development concepts.

---

## Features

- User authentication (job seekers and recruiters)
- Post new job openings
- View and search available jobs
- Apply for jobs
- Responsive user interface

---

## Tech Stack

- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
